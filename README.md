Netflix Movie Data Analysis

-Project Overview
This project analyzes Netflix movies and TV shows dataset using Python.  
The analysis includes data cleaning, visualization, and extracting insights from the dataset.

-Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook

-Dataset
The dataset contains information about Netflix movies and TV shows including:
Title
Genre
Release Year
Rating
Country
Duration

-Analysis Performed
Data Cleaning
Handling Missing Values
Genre Analysis
Country-wise Analysis
Ratings Analysis
Trend Visualization

-Key Insights
Most Netflix content was added after 2015
Drama is the most common genre
USA has the highest number of titles
Movies are more than TV shows on Netflix

-How to Run the Project
1. Clone the repository
2. Install required libraries
bash
pip install -r requirements.txt
3. Open Jupyter Notebook
4. Run all cells
